# Task Manager API

A simple RESTful API built using Flask that allows users to manage tasks.

## Features

- Add tasks (POST)
- View tasks (GET)
- Delete tasks (DELETE)
- JSON-based persistent storage

## API Endpoints

GET /tasks  
POST /tasks  
DELETE /tasks/<id>

## Tech Stack

- Python
- Flask
- JSON

## Why This Project Matters

Demonstrates:
- REST API design
- CRUD operations
- Backend architecture
- Data persistence
